# Command Reference

## Authentication

### `grape login`

Authenticate with the Trellis platform using Device Authorization Grant.

```bash
grape login [--force]
```

| Flag | Description |
|------|-------------|
| `--force` | Re-authenticate even if already logged in |

### `grape logout`

Remove stored credentials.

```bash
grape logout
```

## Vineyard Management

### `grape vineyard create`

Create a new vineyard (workspace).

```bash
grape vineyard create --name <name>
```

| Flag | Required | Description |
|------|----------|-------------|
| `--name` | Yes | Vineyard name (unique per user) |

### `grape vineyard list`

List all vineyards.

```bash
grape vineyard list
```

Displays a table with vineyard ID, name, vine count, and creation date.

### `grape vineyard delete`

Delete a vineyard and all its vines.

```bash
grape vineyard delete --id <vineyard-id>
```

| Flag | Required | Description |
|------|----------|-------------|
| `--id` | Yes | Vineyard UUID |

## Vine Operations

### `grape vine list`

List vines, optionally filtered by vineyard.

```bash
grape vine list [--vineyard-id <id>]
```

| Flag | Required | Description |
|------|----------|-------------|
| `--vineyard-id` | No | Filter by vineyard |

### `grape vine get`

Get detailed information about a vine.

```bash
grape vine get --id <vine-id>
```

| Flag | Required | Description |
|------|----------|-------------|
| `--id` | Yes | Vine UUID |

## Infrastructure Operations

### `grape plan`

Queue a Terraform plan job for a vine.

```bash
grape plan [--vine-id <id>] [--worker-id <id>] [--wait]
```

| Flag | Required | Description |
|------|----------|-------------|
| `--vine-id` | No | Vine UUID (interactive selector if omitted) |
| `--worker-id` | No | Specific Tendril to execute (any available if omitted) |
| `--wait` | No | Block until the job completes |

### `grape harvest`

Queue a deployment (apply) job for a vine.

```bash
grape harvest [--vine-id <id>] [--worker-id <id>] [--plan-job-id <id>] [--wait]
```

| Flag | Required | Description |
|------|----------|-------------|
| `--vine-id` | No | Vine UUID (interactive selector if omitted) |
| `--worker-id` | No | Specific Tendril to execute |
| `--plan-job-id` | No | Reference a specific plan to apply |
| `--wait` | No | Block until the job completes |

If `--vine-id` is omitted, an interactive TUI selector prompts you to choose a vineyard and vine.

### `grape destroy`

Queue a destroy job to tear down infrastructure.

```bash
grape destroy [--vine-id <id>] [--worker-id <id>] [--wait]
```

| Flag | Required | Description |
|------|----------|-------------|
| `--vine-id` | No | Vine UUID (interactive selector if omitted) |
| `--worker-id` | No | Specific Tendril to execute |
| `--wait` | No | Block until the job completes |

## Job Management

### `grape jobs list`

List all jobs.

```bash
grape jobs list
```

Displays a table with job ID, type, status, vine name, and timestamps.

### `grape jobs get`

Get detailed information about a job.

```bash
grape jobs get --id <job-id>
```

| Flag | Required | Description |
|------|----------|-------------|
| `--id` | Yes | Job UUID |

### `grape jobs logs`

Stream job logs.

```bash
grape jobs logs --id <job-id> [--follow]
```

| Flag | Required | Description |
|------|----------|-------------|
| `--id` | Yes | Job UUID |
| `--follow` | No | Continuously stream new log entries (like `tail -f`) |

### `grape jobs cancel`

Cancel a running or queued job.

```bash
grape jobs cancel --id <job-id>
```

| Flag | Required | Description |
|------|----------|-------------|
| `--id` | Yes | Job UUID |

### `grape jobs wait`

Block until a job reaches a terminal state (SUCCESS, FAILED, or CANCELLED).

```bash
grape jobs wait --id <job-id>
```

| Flag | Required | Description |
|------|----------|-------------|
| `--id` | Yes | Job UUID |

## Cluster Management

### `grape clusters list`

List all provisioned Kubernetes clusters.

```bash
grape clusters list
```

Displays cluster name, provider, region, endpoint, and ArgoCD URL.

## Tendril Management

### `grape worker list`

List all registered Tendrils.

```bash
grape worker list
```

Displays Tendril ID, name, status (ONLINE/OFFLINE/DRAINING), mode, version, and last heartbeat.

### `grape worker config`

Show the current Tendril configuration.

```bash
grape worker config
```

### `grape worker remove`

Remove a registered Tendril.

```bash
grape worker remove --id <worker-id>
```

| Flag | Required | Description |
|------|----------|-------------|
| `--id` | Yes | Worker UUID |

## Selectors

### `grape selectors`

Launch interactive TUI selectors for choosing a vineyard and vine.

```bash
grape selectors
```

See [Interactive Selectors](/docs/grape/selectors) for details.
