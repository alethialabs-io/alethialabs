# Platform Overview

This platform is a hybrid multi-cloud infrastructure orchestration system. It combines a visual web interface, a command-line tool, and autonomous execution agents to provision production-grade Kubernetes clusters with databases, caches, networking, DNS, secrets, and GitOps — across AWS, GCP, and Azure.

<img src="/docs/diagrams/concepts/platform-overview.svg" alt="Platform Architecture" />

## Three-Component Model

| Component | Type | Language | Role |
|-----------|------|----------|------|
| **Console** | Web application | TypeScript (Next.js 15) | Visual configuration, job management, real-time monitoring |
| **alethia** | CLI tool | Go | Terminal-based operations, scripting, CI/CD integration |
| **Runner** | Execution agent | Go | Runs Terraform, installs ArgoCD, streams logs back to Console |

## Data Flow

```
User ─► Console (web) ─► Job Queue (PostgreSQL) ─► Runner (agent) ─► Cloud Provider
         │                                            │
         ├─ OR ──► alethia CLI ─► Job Queue ──────────┘
         │
         └─ LISTEN/NOTIFY → SSE ◄── Log Streaming ◄── Runner
```

1. The user configures infrastructure through Console (web form) or alethia (CLI commands)
2. Actions are queued as jobs in PostgreSQL
3. A Runner agent claims the job atomically and assumes temporary cloud credentials
4. The Runner executes Terraform and streams logs back in real-time
5. On completion, ArgoCD is installed for GitOps-managed deployments

## Multi-Cloud Support

Every infrastructure component has provider-specific implementations across three clouds:

| Capability | AWS | GCP | Azure |
|-----------|-----|-----|-------|
| Kubernetes | EKS | GKE | AKS |
| Relational DB | Aurora PostgreSQL/MySQL | Cloud SQL | Azure Database |
| Cache | ElastiCache (Redis/Valkey) | Memorystore | Azure Cache for Redis |
| NoSQL | DynamoDB | Firestore | Cosmos DB |
| Messaging | SQS + SNS | Pub/Sub | Service Bus |
| DNS | Route 53 | Cloud DNS | Azure DNS |
| Secrets | Secrets Manager | Secret Manager | Key Vault |
| Container Registry | ECR | Artifact Registry | ACR |
| Object Storage | S3 | Cloud Storage | Storage Account |
| WAF | AWS WAFv2 | Cloud Armor | Azure WAF |

## Tech Stack

**Web (Console):** Next.js 15, PostgreSQL + Drizzle ORM, Better Auth, LISTEN/NOTIFY + SSE realtime, S3-compatible storage (SeaweedFS), Tailwind CSS, shadcn/ui, Zustand

**CLI (alethia):** Go, Cobra, Charmbracelet (lipgloss, huh, bubbletea), Homebrew tap

**Agent (Runner):** Go, ECS Fargate (ARM64), embeds Terraform + kubectl + Helm + cloud CLIs

**Shared (core):** Go package — provisioner, Terraform wrapper, cloud SDKs, ArgoCD, Infracost, Git, Helm

**Infrastructure:** Terraform, S3-compatible storage / SeaweedFS (state), Lambda scaler (scale-to-zero), release-please

│   ├── charts/           # Helm charts

└── .github/workflows/    # CI/CD (release-please, release-alethia, release-runner)
