# Introduction

Documentation for **Alethia** — the multi-cloud Kubernetes control plane by Alethia Labs. Configure
infrastructure visually in the Console, deploy from the terminal with the `alethia` CLI, and let Runners
execute OpenTofu and reconcile GitOps. Choose a section to get started.

<Callout type="info">
  **Two editions, one codebase.** Alethia is open-core: a free, open-source <Community>Community</Community>
  edition (the complete single-tenant product) and a licensed <Enterprise>Enterprise</Enterprise> edition
  (multi-member organizations, teams, SSO, custom roles, and the fine-grained access portal). See
  [**Editions**](/editions) for the full comparison.
</Callout>
