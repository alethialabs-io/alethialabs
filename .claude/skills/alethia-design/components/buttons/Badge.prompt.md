One-line: Compact label for counts, statuses, versions, and region codes.

```jsx
<Badge variant="solid">3 active</Badge>
<Badge variant="muted">Aurora PostgreSQL</Badge>
<Badge variant="outline" mono>eu-west-1</Badge>
```

Variants: `solid`, `muted` (default), `outline`. Set `mono` for the uppercase mono treatment used on technical tags (region codes, k8s versions, provider IDs).
