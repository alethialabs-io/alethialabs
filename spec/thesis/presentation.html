Сигурност

Zero-Key мулти-клауд архитектура

Платформата не съхранява облачни credentials. Всеки достъп е временен и отменяем.

AWS

Cross-Account IAM Role

CloudFormation шаблон създава IAM роля с External ID. Tendril поема ролята чрез STS AssumeRole. Сесия: 1 час.

GCP

Workload Identity Federation

OIDC токен от Trellis се обменя за временни GCP credentials. Без service account JSON ключове.

Azure

Federated Identity

App Registration с федерирани credentials в Entra ID. OIDC автентикация. Без client secrets.

CLI автентикация: RFC 8628 Device Authorization Flow. Потребителят потвърждава в браузъра — CLI получава Refresh Token, криптиран локално. Без пароли в терминала.

Бъдещо развитие

Наблюдение

Monitoring интеграции

Свързване с Datadog, Grafana и Prometheus. Автоматично конфигуриране на дашборди и аларми при провизиране на нов клъстер.
